# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-05

### Added
- Core runtime with signal propagation and resonance-based activation
- Agent lifecycle management with health tracking and state machines
- Tuning drift for adaptive agent behavior
- PostgreSQL persistence with pgvector for vector similarity
- Validation service for output quality assurance
- HTTP API with SSE event streaming
- Capabilities: Search, Synthesizer, Code Writer, Code Reviewer, Analyst
- Provider support: Anthropic, OpenAI, Ollama (local LLM), Brave Search, Cohere
- CLI with run, serve, status, and configuration commands
- Comprehensive test suite with 72 passing tests
- Database migrations with pgvector support
